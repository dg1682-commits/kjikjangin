📰 실시간 사내 속보 뉴스룸 채널 개설 및 5턴 트렌드 동기화 안내

  • 사내 실시간 뉴스 시스템 도입: 메인 화면 좌측 하단에 고정형 신문 아이콘이 배치되었습니다. 통제실에서 특보가 발행되면 실시간 무결성 검사 엔진을 통해 0.1초 만에 메인 화면에 'N' 알림 배지가 켜집니다.
  • 통제실 5턴 트렌드 제어 기능 탑재: 주식 통제실(chart.html)에서 특정 종목에 강제 호재(폭등) 및 악재(폭락) 뉴스 초안을 자동 빌드하여 송출할 수 있습니다. 송출 시 전체 유저의 시장 배율에 정확히 5턴(7분 30초) 동안 연속 복리 가속이 강제 인젝션됩니다.
  • 기사 자동 만료 및 취소선 시스템: 5턴의 가속 버프/디버프 정산이 끝나면 뉴스룸(news.html)의 해당 카드에 실시간으로 취소선 처리와 함께 [효과 종료] 마크가 안전하게 렌더링됩니다.
  • 과거 데이터 원격 일괄 관리자 추가: 통제실 하단에 과거 기사 일괄 만료 기능 및 뉴스 데이터 공장 초기화(비우기) 버튼 매니저가 배치되어 유연한 운영이 가능해졌습니다.
  • 뉴스룸 UI 최적화 개편: 효과가 종료된 구버전 특보 배너가 상단을 가리고 있던 현상을 수정하여, 배너를 전격 삭제하고 시원하고 신뢰감 높은 프리미엄 언론사 카드 뷰 레이아웃으로 넓게 확장했습니다.

📮 전 사원 대상 원격 '푸시 우편함 시스템' 가동 및 자산 표기 방어 패치

  • 운영자 원격 보상 우편함 탑재: 설정(⚙️) 메뉴 내부에 우편함 기능이 추가되었습니다. 운영자가 닉네임 변경권, 현금 보상, 체력 버프 등을 전 사원에게 실시간으로 원격 지급할 수 있습니다.
  • 자산 1조원 표기 짤림 방어: 현금이 천만 단위를 넘어 억, 조 단위로 올라갈 경우 우측 상단 UI 레이아웃에서 텍스트가 잘려 보이던 현상을 유연 폰트 스케일링 기법(clamp CSS)으로 긴급 패치하였습니다.
  • 통합 마스터 알림 배지 연동: 읽지 않은 우편물이 있거나 새로운 공지사항이 있을 경우 메인 화면의 우측 하단 ⚙️ 설정 버튼 위에 통합 알림 빨간 불이 실시간 동기화됩니다.
